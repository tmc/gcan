NAME
----

gcand − gcan daemon

SYNOPSIS
--------

**gcand** \[OPTIONS\]

DESCRIPTION
-----------

OPTIONS
-------

**−v**, **−−verbose**

Be verbose

**−h**, **−−help**

Show this help message

COMMANDS
--------

**start**
 starts the gcand daemon

**Usage**: gcand \[OPTIONS\] start \[start-OPTIONS\]

**−l**, **−−listen** &lt;default: *":4226"*&gt;

Listen address

**−h**, **−−help**

Show this help message
